# Task Manager App

A simple task management app built with Node.js and TypeScript

## Tech Stack
- Node.js, TypeScript
- Express
- In-memory data storage for demo
- Repository abstraction for easy swapping between storage implementations (mock, Postgres, etc.)

## Features
- User registration and authentication
- Create, update, and delete tasks
- Mark Tasks as Done / not Done

## Quick Start
1. Clone the repo
2. cd task-manager-backend
3. Install Dependencies - npm install
4. Run the project - npm run dev
